For immersive stories

Story by Hadrian Cawthorne

Building a long-form web page

Building your long-form multimedia page is reasonably straight forward. This is a pre-built HTML, CSS and JavaScript framework for you to copy and work with.

With a bit of basic coding knowledge you should be able to pick up this framework with reasonable ease and work with it.

Production Workflow

The framework is hosted on Github at:

  • https://github.com/haddersbadders/Simple-longform-framework

Just download the Zip file and extract. This gives you a working project folder with a copy of this framework.

Edit the files using your favourite code editor. You might consider using:

  • Notepad++ (PC)
  • TextWrangler (Mac)
  • Atom.io (Cross platform)

Edit the index.html, style.css and index.js files in your code editor and save after editing.

Keep the index.html file open in a web browser (Chrome) and refresh the page when you want to see the changes.

Working with media

The img folder is for you to put your images in. Try and keep image sizes small - aim to rezize to 1050px wide of full width images and 800px wide for content images. They should not exceed more than a few hundered KB each.
